yui3-gallery by lsmith

Documentation and examples for YUI 3 Gallery modules contributed by Luke Smith.

  • API docs
  • Examples

Components

gallery-datatable-350-preview
The version of Y.DataTable that will appear in version 3.5.0 of the library, replacing the prior version. This module is a development preview that will be updated through the 3.5.0 development cycle, and will be deprecated on the official release in 3.5.0.
gallery-jsonp
Deprecated. Use the core module).
gallery-torelativetime
Translates Dates into relative time strings, such as "25 minutes ago".
gallery-dataschema-class
Makes Y.DataSchema an instantiable class, allowing you to encapsulate schema configuration in a reusable instance.
gallery-pluginattr
Allows Plugins to register an attribute on host classes that automatically plug or unplug the plugin.
gallery-textnodes
Adds support for node.get('textNodes') to return an array of text nodes (akin to node.get('children')).
gallery-jsonrpc
Provides a Y.JSONRPC class to encapsulate a remote API that supports the JSON-RPC transport. Also adds Y.jsonrpc(url, method, paramArray, callback) method.
gallery-event-konami
Adds a "konami" DOM event (node.on('konami', activateGodMode)) that is triggered when users enter the Konami code.
gallery-event-drag
(Abandoned) Adds syntehtic events to allow subscribing to "drag", "drag:start", "drag:end". Subscribing to these events will plug the Node with Y.Plugin.Drag.
gallery-event-nav-keys
(Mostly subsumed by event-key and gallery-event-arrow) Adds synthetic events "enter", "esc", "backspace", "tab", "pageUp", "pageDown", "up", "down", "left", and "right" (arrow keys).
gallery-event-arrow
Adds an "arrow" synthetic event. The event object includes a "direction" property that will be one of "n", "s", "e", "w", "nw", "ne", "sw", or "se".
